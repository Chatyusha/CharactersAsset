# CharactersAsset

These are files that lists the characters used in each country's language.

These will be useful when you use TextMeshPro.

## Supported Languages

- Japanese
  - Kanji (from UnicodeCJK)
  - Hiragana
  - Katakana
  
- Chinese
  - Kanji (from UnicdeCJK)
 
- Korean
  - Kanji (from UnicodeCJK)
